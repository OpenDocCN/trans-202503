## 关于作者

Corey Ball 是 Moss Adams 的网络安全咨询经理，负责领导渗透测试服务。他在 IT 和网络安全领域拥有超过 10 年的工作经验，涵盖多个行业，包括航空航天、农业、能源、金融科技、政府服务和医疗保健。除了获得萨克拉门托州立大学的英语和哲学学士学位外，他还持有 OSCP、CCISO、CEH、CISA、CISM、CRISC 和 CGEIT 等行业认证。
