# 附录 B. 与 GNS3 兼容的 Cisco 硬件

本附录列出了 Dynamips 和 GNS3 支持的 Cisco 硬件。Dynamips 仅模拟使用 MIPS 架构的 Cisco 路由器，并不支持所有 Cisco 型号。在支持的路由器列表之后，你会看到一份推荐的 IOS 镜像文件列表，GNS3 可以自动应用正确的 Idle-PC 值，而无需执行 Idle-PC 计算。

# 支持的 Cisco 硬件
